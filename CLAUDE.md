# CLAUDE.md: UnB-SAT `.github`

This repo is the **static** organization profile shown at <https://github.com/UnB-SAT>
(from `profile/README.md`): group overview, research lines, the principal
investigator, and a link to the website. Edit `profile/README.md` directly; nothing
here is generated, and there is no data or build script in this repo.

The website (team, theses, undergraduate research, publications) and the
`data/students.yml` + generator that build it live in the sibling repository
**`unb-sat.github.io`** (<https://unb-sat.github.io>). See that repo's `CLAUDE.md`
for the data and generator workflow.

## Conventions

- Body text in English; no em dash (`—`); few emojis; researcher tone.
- Keep the profile slim: overview, research lines, the PI card, and the website link.
  Do not duplicate the team, theses, IC or publications here; those live on the site.
- The PI card and the website link are plain HTML/Markdown here. If they change, edit
  by hand and keep them consistent with the site.
